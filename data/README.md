# Dados

As imagens não são armazenadas neste repositório.

Consulte [`../docs/DATASETS.md`](../docs/DATASETS.md) para:

- conhecer as bases públicas previstas;
- acessar os links oficiais;
- entender os requisitos de uso;
- organizar as pastas locais.

Estrutura sugerida:

```text
data/
├── montgomery/
├── shenzhen/
├── chexpert/
└── mimic-cxr-jpg/
```

## Regras

- Não coloque radiografias privadas ou identificáveis nesta pasta.
- Não faça commit das bases públicas completas.
- Respeite os termos e licenças de cada fonte.
- Registre a versão e a data de download.
- Revise metadados e textos gravados nos pixels.
